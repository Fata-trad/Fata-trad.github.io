"Dreams of the Revenant" est une version remasterisée du roman visuel "The House In Fata Morgana". Cette version ajoute egalement de nouveaux contenus au jeu original, notamment de nouvelles histoires, de nouveaux personnages et des scènes supplémentaires.

Le patch est développé pour le jeu Seventh Lair, qui est disponible sur Steam. Par conséquent, afin d'installer le patch, il est impératif de posséder le jeu Seventh Lair au préalable
+

S'il vous plaît, achetez The House in Fata Morgana Dreams of the Revenant Edition ainsi que le jeu Seventh Lair sur Steam pour soutenir Novectacle.



Étape 1 : Préparation des fichiers

1. Assurez-vous d'avoir téléchargé le fichier "com.NOVECT.TheHouseinFataMorgana.rar" et "FatamoruAndroid-VX.X.apk" sur votre appareil Android.


Étape 2 : Extraction du fichier rar

1. Téléchargez une application de gestion de fichiers depuis le Play Store si vous n'en avez pas déjà une.
2. Ouvrez l'application de gestion de fichiers et naviguez jusqu'au dossier où vous avez téléchargé le fichier "com.NOVECT.TheHouseinFataMorgana.rar".
3. Appuyez longuement sur le fichier, puis sélectionnez "Extraire" ou "Décompresser" dans le menu contextuel.
4. Choisissez le dossier "android/data" comme emplacement de destination pour extraire les fichiers.
5. Assurez-vous que le dossier extrait se trouve maintenant dans le dossier "android/data".


Étape 3 : Installation de l'APK

1. Retournez dans le dossier où vous avez téléchargé le fichier APK
2. Appuyez sur le fichier "FatamoruAndroid-VX.X.apk" pour lancer l'installation. (Vous serez peut-être invité à activer l'option "Sources inconnues" dans les paramètres de sécurité de votre appareil Android. Si tel est le cas, suivez les instructions à l'écran pour activer cette option. Cela permet d'installer des applications provenant de sources autres que le Play Store.)
3. Suivez les instructions à l'écran pour terminer la procédure d'installation du jeu.


Félicitations ! Vous avez maintenant installé et configuré le jeu "The House in Fata Morgana" sur votre appareil Android. Vous pouvez maintenant profiter du jeu en lançant l'application à partir de votre liste d'applications ou de l'écran d'accueil.
